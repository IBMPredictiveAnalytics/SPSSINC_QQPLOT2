SPSSINC QQPLOT2 Extension Command

Create a QQ plot of one variable against another or a plot of the same variable in a partition of the data into two subsamples.

SPSSINC QQPLOT2 VARIABLES=one or two variable-names*
BY=variable
OPTIONS TITLE=“text”

* Required

SPSSINC QQPLOT2 /HELP prints this information and does nothing else.

SPSSINC QQPLOT2 VARIABLES=mpg BY=type.

Split files and weight are not honored by this command.

VARIABLES specifies one or two variable names. If there are two, their distributions are compared. The y variable comes first. If there is one, a grouping variable must be specified, and the groups defined by that variable will be compared.

BY specifies a grouping variable, which should have exactly two values. If there are more, the first two will be used.

OPTIONS

TITLE can be specified as a quoted string to replace the default title.

PROGRAMFILE is no longer supported.
